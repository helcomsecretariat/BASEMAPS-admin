About Basemaps

Basemaps provides access to maritime spatial planning (MSP) data in the Baltic Sea. It is designed for planners, data providers and authorities involved in maritime spatial planning.

Open the BASEMAPS user guide for more detailed instructions on how to use this portal.

The MSP input and output data terminology is used according to the HELCOM-VASAB MSP Data expert sub-group to distinguish between the data used in the development of the plans and the plans themselves.

MSP input data: This section is a valuable resource for the data used in the development of the plans. It includes data and metadata published and maintained at source by national data providers. Data are available as OGC WMS, OGC WFS, ArcGIS REST Map Servers or provided as data packages for download. Section also includes harmonized and centralized Baltic-wide datasets provided by HELCOM.

MSP output data: This section is useful to examine the existing plans of Baltic Sea countries. National data in both sections are added and maintained by national data providers.
